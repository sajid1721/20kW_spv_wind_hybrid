# 20 kW SPV-Wind Hybrid Power Plant – Bhola, Bangladesh

## 📌 Project Overview
This project presents the design of a **20 kW grid-connected hybrid renewable energy system**, integrating **solar photovoltaic (SPV)** and **wind energy** to supply electricity to a rural community in Char Kukrimukri, Bhola, Bangladesh.
The system ensures reliability by combining solar and wind, taking advantage of Bhola’s high solar irradiance and moderate coastal wind speeds.

## 🌍 Location
📍 Char Kukrimukri, Bhola
The site was selected due to:
- Good average solar radiation throughout the year
- Coastal wind resource availability
- Limited grid stability in rural households
